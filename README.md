This repository contains my personal Linux rice configuration.

## Components

- i3 (window manager)
- i3blocks (status bar)
- Rofi (application launcher)
- Picom (compositor)
- Custom scripts (wifi.sh, etc.)
- Required fonts: Font Awesome, Noto Serif

## Installation

1. Install required packages:
   - i3
   - i3blocks
   - rofi
   - picom
   - qt5
   - fonts (Font Awesome, Noto Serif)

2. Copy configs:
   cp -r i3 ~/.config/
   cp -r rofi ~/.config/
   cp -r picom ~/.config/

3. Reload i3:
   Mod + Shift + R

## Notes

- Designed for Debian/Ubuntu-based systems.
- Fonts are required for icons to render correctly.
